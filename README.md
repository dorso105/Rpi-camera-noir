# Rpi-camera-noir
Code and 3d models for a night vision camera. with list of hardware and code.

Hardware includes 
1. Pi zero (pi zero W can be used but isnt cost effective) 
2. Rpi camera module 3 noir (needs to be noir)
    Can be other camera modules made by Raspberry pi but has to have a noir filter. [See this image from RPI forums on why this is](https://stamm-wilbrandt.de/en/forum/bulb.irled.anim.gif)                                    
3. 3d files and some [fillament](https://au.store.bambulab.com/collections/bambu-lab-3d-printer-filament?Printers++Type=For+A1+Series). (oh and a printer) PLA works well. If using outside PETG is more suitible and can be printed using A1 series bambu printers     
       suggested fillament with a 69°c heat resistance for A1 printers [petg for a1](https://au.store.bambulab.com/products/petg-hf)                                                                                             
4. LCD screen. Stl files support; [lcd screen](https://core-electronics.com.au/13inch-ips-lcd-display-hat-240x-240-pixels.html)                                                                                                                                 
5. [Battery](https://core-electronics.com.au/3-pin-lipo-battery-for-pijuice-zero-1000mah.html) if using the device without powersupply via cord 
       can use pi zero juice but files will not support it. - I am working on files to fit the [juice hat](https://core-electronics.com.au/waveshare-ups-hat-for-pi-zero.html)
6. scews that fit hardware (m2.5)  Heated thread inserts are needed.
7. **Stl files need to be upsized by 1% in slicer.**
